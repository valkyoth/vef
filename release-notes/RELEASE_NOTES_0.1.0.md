# VEF 0.1.0 Release Notes

Status: planned; not released

VEF 0.1.0 establishes the repository and review boundaries only. It does not
yet parse, serialize, or serve HTTP.

## Planned contents

- Dependency-free `no_std` workspace and facade.
- MIT OR Apache-2.0 licensing.
- Pinned Rust 1.97.1 with Rust 1.90.0 MSRV.
- Security, conformance, RFC provenance, portability, and release policies.
- CI, tooling freshness checks, RFC integrity checks, and pentest workflow.

## Security

No protocol support is claimed. A passing exact-commit pentest report is still
required before tagging `v0.1.0`.
